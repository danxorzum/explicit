# Explicit Example

A Very Good CLI Dart app demonstrating the `explicit` Result package.

## Quick path

```sh
dart pub get
dart run
```

The default command prints six sections: success/failure, `flatMap`/`andThen`,
deprecated `toRecord`, lazy `AsyncRes`, `AsyncRes` with `retry`, and awaited
composition.

## Commands

```sh
dart run
dart run explicit_example demo
dart run explicit_example --version
```
